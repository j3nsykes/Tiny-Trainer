Real-time gesture recognition from Arduino + Tiny Trainer

How it works:

  • Train your Tiny Trainer model with gestures (circle, figure8, etc.)
  • Upload the sketch to Arduino with sensor streaming
  • Run Serial Bridge: npm start
  • Perform gestures and watch the visualization!

Device Configuration:

This sketch expects predictions from arduino_1

To change the device name, edit the bridge.onData('arduino_1', ...) line in sketch.js
